# leetcode算法详述

## 一、简介
一个可以方便自己刷题的工程，包含最佳答案，空白题描述，以及按分类刷题


## 二、目录结构
目录结构如下
-leetcode-algo/src/main/java/arthur/dy/lee
 - algo 算法
   - answer 最佳答案
   - blank 空题目，刷题用
   - classify 题目分类
     - backtracking 回溯
     - binary_tree 二叉树
     - dynamic_programming 动态规划
     - greedy 贪心
     - linked_list 链表
     - matrix 矩阵
     - recursion 递归
     - stack 栈
     - string 字符串
     - ... ...
   - common 常用刷题工具类，比如ListNode&TreeNode
 - mysql 数据库
 - pattern 设计模式
